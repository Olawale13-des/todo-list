# 📝 To-Do List Web App

A simple and clean To-Do List web app built using **HTML**, **CSS**, and **JavaScript**.

## 🚀 Features

- ✅ Add new tasks
- 🟢 Mark tasks as completed
- ❌ Remove tasks by double-clicking
- ✨ Responsive and clean UI
